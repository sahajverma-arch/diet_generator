# Public assets

## Brand logo (optional but recommended)

Drop your LEANR logo here as **`logo.png`**:

```
public/logo.png
```

- It is embedded on the **cover page** of every generated PDF.
- Recommended: a horizontal, transparent-background PNG, roughly **520 × 160 px**
  (it is scaled to ~132 × 40 pt on the cover).
- Prefer the **white/yellow** version of the mark, because the cover background
  is black.

If `logo.png` is absent, the PDF and UI automatically fall back to the
typographic **"LEANR."** wordmark, so the app still works without it.
